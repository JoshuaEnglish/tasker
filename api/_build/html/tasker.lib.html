list_tasks([by_pri, filters, filterop, showcomplete, showuid)[source]¶

Displays a list of tasks.

Parameters

  • by_pri (bool) – If true, sorts by priority, if false, sorts by order in file

  • filters (str) – Words to filter the list

  • filterop (func) – all or any (the functions, not strings

  • showcomplete (bool) – If true, shows completed tasks

  • showext (bool) – If true, shows the normally hidden extensions of the task.

  • opendate (date) – If not None, filters tasks opened on opendate

  • closedate (date) – If not None, filters tasks closed on closedate

  • hidedate (date) – The date to filter extensions marked to hide

Return type

TASK_OK, None
